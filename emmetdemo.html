Emmet Doc

Child

Child : >
nav>ul>li 꺽괄호로 계층설정

Sibling

Sibling : +
두 개 이상의 요소를 합쳐 형제 요소를 만듦
div+p+bq

Climb-up

Climb-up : ^
div+div>p>span+em^bq
span+em이 p안에서 생성되지만, p 밖에 bq를 만듦(상승이동)

Multiplication

Multiplication : *
ul>li*5 반복생성

Numberling

Numberling : $
ul>li.item$*5
item뒤에 $자리에서 번호를 1~5까지 생성
